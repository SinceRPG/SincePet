Added Native Upgrading Points System & Base/Max-Value Support (#3)

Features:
- Native Upgrading Points System: Pets now natively earn upgrading points as they level up, which can be spent directly in their skill tree without relying on external economy plugins.
- Configurable per Pet: Fine-tune point generation per pet via the `upgrading_points` block (`levels_per_point`, `points_per_interval`, `max_points`).
- Stat Limits: Added `base` and `max-value` configuration parameters for every stat calculation to give fine-grained control over balancing.
- Documentation: Completely updated GUI, Upgrades, and Pets documentation.
- Placeholder Support: Added `<upgrading_points>` and `<max_upgrading_points>` GUI placeholders.

Important Note for Server Owners:
Please BACKUP your `plugins/SincePet` folder and DELETE the old folder to let the plugin regenerate the new fully-configured pet files, which include the out-of-the-box native upgrading point system setups.
